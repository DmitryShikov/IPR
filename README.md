# IPR — Исследование производительности и диагностика Java-приложений

Данный репозиторий содержит набор практических работ по исследованию работы JVM, профилированию Java-приложений и анализу проблем производительности.

Основная цель проекта — изучение инструментов мониторинга, профилирования и диагностики памяти на практических примерах.
